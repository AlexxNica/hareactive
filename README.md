# hareactive
Experimental FRP library for building web applications.


# Contribution

```
npm install
```

## Run tests

```
npm test
```

## Benchmark
To run a single benchmark: (You will have to compile some of the files first)
```
node benchmark/<name-of-benchmark>
```


example:
```
node benchmark/scan
```


To run all benchmarks
```
npm run bench
```
